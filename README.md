# MERN Vite Project

This is a full-stack web application built using the **MERN stack** ( Express, React, Node.js) with **Vite** for blazing-fast frontend development.

---

## 🌐 Tech Stack

### Frontend
- ⚛️ React 19
- ⚡ Vite
- 🧠 ESLint
- 📝 React Markdown & syntax highlighting (PrismJS / Highlight.js)

### Backend
- 🧰 Node.js
- 🚂 Express
